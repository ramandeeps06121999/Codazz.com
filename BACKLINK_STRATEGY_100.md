## 📋 BACKLINK TRACKING SPREADSHEET

Create a Google Sheet with these columns:

| Column | Description |
|--------|-------------|
| ID | Unique identifier (001-100) |
| Date Submitted | When you submitted/applied |
| Source Name | Website/publication name |
| Source URL | Full URL |
| Source DA | Domain Authority (use Ahrefs/SEMrush) |
| Link Type | Directory/Guest Post/HARO/Partnership |
| Status | Pending/Approved/Live/Rejected |
| Date Live | When link went live |
| Link URL | URL where your link appears |
| Anchor Text | Text used for the link |
| Landing Page | Which page on codazz.com |
| Notes | Any additional information |
| Follow-up Date | When to follow up |

### Status Definitions:
- **Pending:** Submitted, waiting for response
- **In Progress:** Communication ongoing
- **Approved:** Accepted, waiting for publication
- **Live:** Link is published and active
- **Rejected:** Declined or no response after follow-up
- **Lost:** Previously live link removed

## ⚠️ AVOID THESE LINK BUILDING MISTAKES

### ❌ Black Hat Tactics (Will Get You Penalized)
- Buying links from link farms
- Private Blog Networks (PBNs)
- Automated link building software
- Comment spam
- Forum signature spam
- Exact-match anchor text over-optimization
- Link exchanges ("you link me, I link you")
- Hidden links

### ❌ Low-Quality Links (Waste of Time)
- Directories with no editorial process
- Article syndication sites (EzineArticles, etc.)
- Social bookmarking sites
- Web 2.0 profiles with no content
- Foreign language sites (unless relevant)
- Sites with DA < 20

### ✅ White Hat Only
- Editorial links from real publications
- Guest posts on relevant blogs
- Natural directory listings
- HARO/media mentions
- Resource page links
- Partnership links
- Award/recognition links
