## How to Use These URLs

### Option 1: Direct Hotlink (Not Recommended)
```html
<img src="https://images.unsplash.com/photo-xxx?w=1200" alt="Description">
```
⚠️ Unsplash may rate-limit or change URLs

### Option 2: Download and Host (Recommended)
```bash
# Download image
curl -o hero-image.jpg "https://images.unsplash.com/photo-xxx?w=1920"

# Optimize
# Use TinyPNG.com or Squoosh.app to compress

# Upload to your CDN
# Use WebP format for better compression
```

### Option 3: Next.js Image Optimization
```jsx
import Image from 'next/image';

<Image
  src="/images/blog/hero-image.webp"
  alt="Team working on app development"
  width={1920}
  height={1080}
  priority
/>
```

## Required: Photographer Credit

For every Unsplash image, you MUST include attribution:

```html
<figure>
  <img src="image.jpg" alt="Description">
  <figcaption>
    Photo by <a href="https://unsplash.com/@username">Name</a> on 
    <a href="https://unsplash.com">Unsplash</a>
  </figcaption>
</figure>
```

Or in CSS:
```css
.image-credit {
  font-size: 12px;
  color: #666;
  margin-top: 8px;
}
```

## Image Optimization Tools

1. **Squoosh** (https://squoosh.app) - Google's image optimizer
2. **TinyPNG** (https://tinypng.com) - PNG/JPG compression
3. **ImageOptim** (Mac) - Desktop optimizer
4. **Sharp** (Node.js) - Programmatic optimization

### Recommended Settings:
- Format: WebP (with JPEG fallback)
- Quality: 80-85%
- Hero: 1920x1080, < 200KB
- Content: 1200x800, < 150KB
- Thumbnails: 400x300, < 50KB

```bash

# Save as download-images.sh

mkdir -p blog-images

cd blog-images

# Post 1
curl -o post1-hero.jpg "https://images.unsplash.com/photo-1522071820081-009f0129c71c?w=1920&q=80"

```

Run: `chmod +x download-images.sh && ./download-images.sh`
